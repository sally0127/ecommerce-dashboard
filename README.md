## Ecommerce Dashboard

使用React框架開發的電商分析Dashboard，串接DummyJSON API顯示商品資料。

## 功能

- Dashboard : 顯示商品總數、平均價格、平均評分
- 訂單頁面 : 搜尋訂單、按照價格排序、狀態顏色標示
- 商品頁面 : 顯示圖片、按分類篩選、點擊查看詳細資訊

## 完整配套的技術組合

- React
- React Router
- Vite
- DummyJSON API
